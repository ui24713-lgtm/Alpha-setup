# Alpha Setup

A crypto spot market scanner that helps traders discover high-quality price action setups automatically.

## MVP Features

- Live Binance Spot market data
- Live Bitget Spot market data
- Scan all spot markets
- Pattern detection:
  - Head and Shoulders
  - Inverse Head and Shoulders
  - Double Top
  - Double Bottom
  - Cup and Handle
- Alpha Score
- Timeframe selection
